# Get Started

Welcome to **PlantSCHub**, a community-curated hub for plant single-cell analysis.

This page is the best starting point for new users. Instead of browsing all resources at once, begin with the path that matches your background, your data type, or the question you are trying to answer.

## What is PlantSCHub?

PlantSCHub brings together practical resources for plant single-cell and single-nucleus research in one place. You can use it to find:

- databases and datasets
- protocols and analysis pipelines
- publications
- visualization tools and atlas portals
- tutorials and videos
- contribution pathways for community updates

## Who is this for?

**Experimental biologists**  
Use PlantSCHub to understand what resources exist, find representative studies, and identify practical workflows.

**Computational beginners**  
Use PlantSCHub to learn the basic analysis landscape and find step-by-step resources for common data types.

**Experienced analysts**  
Use PlantSCHub as a fast navigation hub for datasets, code, protocols, and references.

## Before you begin

Before choosing tools or protocols, it helps to know:

- **your data type**: scRNA-seq, snRNA-seq, snATAC-seq, spatial, or multi-omics
- **your species**: some resources are species-specific
- **your genome and annotation version**: especially important for chromatin accessibility workflows
- **your goal**: cell-type annotation, marker discovery, trajectory analysis, regulatory inference, spatial patterning, or cross-species comparison

## Your first 30 minutes on PlantSCHub

If you are new, use this order:

### 1. Identify your data type
Decide whether your project is mainly about gene expression, chromatin accessibility, spatial profiling, or multiple modalities.

### 2. Open the most relevant page
Use the page that best matches your immediate need:

- [Databases](databases) for datasets, markers, and atlas-style resources
- [Protocols](protocols) for pipelines and code resources
- [Publications](publications) for representative studies
- [Tutorials & Videos](videos) for guided learning materials
- [Visualization](visualization) for tools and portals to explore results

### 3. Review one representative example
Pick one database, one protocol, or one publication that matches your species or data type. This helps set expectations before you begin analysis.

### 4. Save useful pages and come back as needed
PlantSCHub is meant to be a navigation hub. Most users will move between pages several times as their project develops.

## Recommended starting points

If you are unsure where to begin, use one of these shortcuts:

- **I only need datasets** → [Databases](databases)
- **I only need analysis resources** → [Protocols](protocols)
- **I want example studies first** → [Publications](publications)
- **I prefer videos and talks** → [Tutorials & Videos](videos)
- **I want to explore outputs and atlas-style tools** → [Visualization](visualization)

## Common questions

### Is PlantSCHub an analysis platform?
No. PlantSCHub is a resource and learning hub that helps users discover relevant tools, workflows, datasets, publications, and training materials.

### Which page should I open first?
If you are new, start with this page and then choose the path that matches your data type or goal.

### I only want datasets or marker resources. Where should I go?
Go to [Databases](databases).

### I only want code or reusable workflows. Where should I go?
Go to [Protocols](protocols).

### I found a missing resource. Can I suggest it?
Yes. Please visit [Contribute](contribute).

## Contribute to PlantSCHub

PlantSCHub grows through community input. If you know of a useful database, protocol, publication, visualization platform, or tutorial that should be included, please help improve the site through the [Contribute](contribute) page.
