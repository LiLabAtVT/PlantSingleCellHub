# Visualization

This page collects tools and portals for exploring plant single-cell data and presenting results.

## When should I use this page?

Use this page if you want to:

- explore atlas-style portals
- inspect clusters, embeddings, or marker patterns
- compare common analysis environments
- see how results are typically visualized in single-cell studies

## How to use this page

A simple way to use these resources:

1. Start with an atlas portal if you want to explore biological patterns quickly.
2. Use Seurat or Scanpy when you need analysis workflows that also support visualization.
3. Return to [Publications](publications) to see how these visual outputs are used in real studies.
