# Databases

This page collects plant single-cell databases, atlas portals, and marker resources.

## When should I use this page?

Use this page if you want to:

- find plant single-cell datasets
- compare available species coverage
- look up marker genes or annotated cell types
- explore atlas-style portals before starting an analysis

If you are new to PlantSCHub, this is often the best place to start when you need **data or biological context** rather than software.

A plant-focused scRNA-seq database curating 67 datasets from 367 experiments across 17 species, covering 2,546,778 cells, 259 cell types, and 229,551 cell markers.

:link-type: doc

A plant single-cell transcriptome database across 33 species, covering 7 tissues, and 58,846 marker genes for 193 cell types.

Added: May 7, 2025
:::

:::{grid-item-card} PsctH
:link: http://jinlab.hzau.edu.cn/PsctH/

Plant Single Cell Transcriptome Hub (PsctH) provides a comprehensive resource of cell markers with experimental evidence from RNA in situ hybridization or GFP reporters, covering 51 cell types across 9 tissues in 5 plant species.

Added: March 5, 2026

## How to use these resources

A practical way to use this page:

1. Start with the database that best matches your species.
2. Check whether annotated cell types or marker genes are available.
3. Use the linked publications or portals to understand how the data were generated and analyzed.
4. Return to [Protocols](protocols) if you need analysis tools.
